=============================================================
  STARTUP MASTER v1.0 - Ultimate Windows Startup Manager
=============================================================

QUICK START:
1. Run StartupMaster.exe (or Launch-StartupMaster.bat)
2. Click "Yes" for administrator elevation
3. Start managing your startup items!

REQUIREMENTS:
- Windows 10 (1809+) or Windows 11
- .NET 8.0 Desktop Runtime (if using framework-dependent build)
- Administrator privileges (for system-wide changes)

FEATURES:
✓ Manage ALL startup locations (Registry, Folders, Task Scheduler, Services)
✓ Enable/disable items (non-destructive)
✓ Add/remove/edit startup items
✓ Configure startup delays
✓ Auto-backup before changes
✓ Performance analysis
✓ Export/import configurations
✓ Modern dark UI
